# og_diag

Playground for data Visualization


## Usage

```
conda activate og_diag
```

## Requirements

To create environment, do

```
conda env create -f env.yaml
```

Activate using

```
conda activate og_diag
```

## Development Environment

To use Jupyter notebook

```
conda activate og_diag
jupyter notebook
```

To use Spyder

```
conda activate og_diag
spyder -p .
```
